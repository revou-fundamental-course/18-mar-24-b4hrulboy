Profil Saya


Pofessional IT

Saya lulusan S1 Teknik Informatika yang memiliki pengalaman kerja lebih dari 6 tahun dibidang IT Support. Memiliki kemampuan pada IT Operation, Troubleshooting Hardware dan Software, Manajemen Database, pengelolaan Jaringan, dan Programming. Salah satu pencapaian saya dalam bekerja, adanya optimalisasi sebanyak 85% pada perangkat/infrastruktur IT dalam kondisi baik dan berfungsi secara optimal. Dalam bidang Database, saya biasa menerapkan system backup secara manual ataupun time schedular. Dalam bidang jaringan, untuk meminimalisir downtime, saya menerapkan alert system sebagai Langkah awal pendeteksian. Selain itu dalam bidang programming , bersama tim saya pernah membuat dan mengembangkan aplikasi seperti aplikasi Portal, Tapping, recruitment, knowledge product test, live chating

Saya menyukai tantangan dan suka belajar hal baru, terutama dalam bidang IT Networking, Programming, Database Administrator, Design

motto saya : jangan pernah putus semangat, terus belajar dan berjuang untuk orang-orang yang kita cintai
